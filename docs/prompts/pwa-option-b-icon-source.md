# Pordee Option B PWA Icon Prompt

Use case: logo-brand
Asset type: production source image for PWA app icon set

Primary request:
Create a polished square PWA app icon source image for `พอดี / Pordee`. The
accepted output is saved as `assets/logo/pordee-option-b-pwa-icon-concept.png`
and resized into all app icon assets, so it must be clean, centered, and
readable at small sizes.

Source mark description to preserve:

- Compact Option B Negative Space PD monogram.
- Coral rounded P loop with vertical left stem and top loop, matching a
  friendly P shape.
- Separated coral right-side D continuation stroke, smaller than the P loop,
  with rounded terminals and clear white space between it and the P.
- Teal balance smile/path tucked below the inner counter, centered under the
  P/D form.
- One tiny lime milestone dot near the lower-right of the mark.
- Keep the PD mark proportions compact, friendly, rounded, not oversized, not
  heavy, not bank-like.

Composition/framing:

Square app icon only. Center the PD mark on a soft sky-tinted rounded-square
tile. The full colored mark should occupy about 58-62% of the icon width and
height. Keep at least 20% visual safe-zone padding on every side, so iOS,
Android, and PWA maskable crops do not clip any coral, teal, or lime element.
No wordmark, no Thai text, no English text, no board layout.

Style/medium:

Premium semi-flat vector-like icon, crisp edges, clean rounded geometry, subtle
tile depth only, app-store ready. The tile may have a very soft sky gradient
and gentle shadow, but the logo mark itself must stay simple and logo-like.

Color palette:

Coral `#FF6B5A`, teal `#18A999`, tiny lime `#B7F34A`, sky `#EAF7FF`, white
`#FFFFFF`.

Constraints:

No wallet icons, piggy banks, coins, coin stacks, charts, bank buildings,
investment symbols, mascot, extra letters, decorative sparkles, glossy 3D,
inflated gel, heavy shadow, watermark, or text. Keep lime as a tiny milestone
accent only. Do not add outlines or black strokes to the mark.
