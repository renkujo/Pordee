# Pordee Semi-Flat PD Logo Prompt Set

Use these prompts one asset at a time. The final selected visual direction is `app/public/logo/pordee-pd-logo.png`.

## 1. Standalone Transparent / Cutout Source

```text
Use case: logo-brand
Asset type: standalone semi-flat logo mark source for transparent PNG extraction
Recommended output: square 1:1, high resolution, centered logo only

Create a single standalone Pordee logo mark only, centered, with generous safe padding, on a perfectly flat solid #00ff00 chroma-key background for background removal.

The logo is a semi-flat rounded P + D monogram for a Thai personal finance app named Pordee. Build it from clean rounded flat shapes: a softened coral lowercase-style P loop on the left, a smaller rounded open D-like coral curve on the right, one muted teal curved smile/balance stroke underneath, and one small pale lime dot near the lower-right end of the mark.

Match the selected reference direction: minimal, warm, friendly, calm, product-ready, vector-friendly. Use smooth rounded shapes with only very subtle tonal softness. No 3D volume, no inflated gel, no clay material, no bevels, no raised tube effect, no specular highlights, no plastic shine, no cast shadow.

Color palette: softened coral #F76A5D for the P and D strokes, muted teal #129C91 for the smile/balance stroke, pale lime #D6E86F for the tiny dot. Do not use the chroma-key green anywhere in the logo.

Composition: no wordmark, no Thai text, no English text, no app icon tile, no card, no border, no mockup board, no mascot, no extra objects. The whole logo must fit comfortably inside the frame with equal padding on all sides.

Background constraints: background must be exactly one uniform flat #00ff00 color, with no shadows, gradients, texture, lighting variation, floor plane, reflections, or transparency checker pattern.

Avoid: 3D, glossy gel, plastic shine, bevel, shadow, wallet icon, piggy bank, coin stack, chart icon, bank logo, extra dots, extra strokes, text, labels, watermark, black outline, hard geometric SVG look, dark cast shadow, floor shadow, cropped edges.
```
